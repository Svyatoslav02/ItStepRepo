POST /api/generate
### Description
This endpoint accepts a text description (**prompt**) and returns a set of images that match the request.  
Currently, the implementation returns static mock data for testing purposes.
---
## Request
### Body
```json
{
  "prompt": "minimalism, japanese style, beige green tones"
}
##Responce
##Succes 200
{
  "prompt": "minimalism, japanese style, beige green tones",
  "images": [
    {
      "url": "https://example.com/image1.jpg",
      "title": "Minimalist interior",
      "sourceUrl": "https://example.com"
    }
  ]
}

prompt (string) — Echoes the input request.
images (array) — A list of image objects.
    url (string) — Direct link to the image.
    title (string) — Title or description of the image.
    sourceUrl (string) — Source of the image.
    
##Error Responses
  Validation Error (400 Bad Request)
  {
    "error": "Prompt field is required"
  }

##Usage Examples

#Request
