# Frontend

React-based client application for the Moodboard AI project.

## Tech stack
- React
- Vite
- React Router

## Project structure
- `moodboard-frontend/` – the React application (source code, components, pages)

## Getting started

```bash
cd moodboard-frontend
npm install
npm run dev
```

## Build

```bash
cd moodboard-frontend
npm run build
```
